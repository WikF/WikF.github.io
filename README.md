# Data CEE - Landing Page

A modern, static landing page for a Data and AI specialized software house.

## Hosting on GitHub Pages

This site is built to be hosted statically.

1.  Push this repository to GitHub.
2.  Go to **Settings** > **Pages**.
3.  Under **Build and deployment**, select **Source** as `Deploy from a branch`.
4.  Select your `main` branch and `/ (root)` folder.
5.  Click **Save**.

## Files
- `index.html`: Main structure.
- `style.css`: Custom animations.
- `script.js`: Carousel logic.
- `.nojekyll`: Config file to bypass Jekyll processing (ensures all assets load).
